Use the HASHCORE widget to drive traffic to your social media networks.

== Description ==

This plugin allows WordPress users to automatically integrate their social media content from Twitter, Instagram etc. at the footer of their articles.

Each publisher has a dedicated dashboard to see the number of clicks through their social media networks.

**IMPORTANT: You will FIRST need to register as a publisher, go to [HASHCORE - sign up for FREE with instant approval](http://hashcore.com/publishers-sign-up-form) and create a HASHCORE account to get your publisher ID to activate the plugin. 

If you have any questions or feedback, please do not hesitate to [contact us](http://hashcore.com/contact/)

== Installation ==

1. Register to get your publisher ID, go to [HASHCORE - sign up for FREE with instant approval](http://hashcore.com/publishers-sign-up-form).
2. Go to the 'Add New' plugins screen in your WordPress admin area and search for 'Hashcore' then click 'Install Now' and activate the plugin.
3. Go to Setting -> Hashcore Settings and follow the instructions.
